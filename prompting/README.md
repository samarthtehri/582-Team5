## Environment

```sh
sh setup.sh
```

## How to Run?

```sh
sh sh/run_llms.sh
```

## Prompts

Prompts are defined in `src/prompts.py`. You can add new prompts and update `get_prompt_template` to use them in `src/run_llms.py`

Please refer to `sh sh/run_llms.sh` for how to use `src/run_llms.py`.
